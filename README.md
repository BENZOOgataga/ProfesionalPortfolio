# Portfolio Professionnel - Louis Morice

Portfolio personnel développé dans le cadre du **BTS SIO option SISR** à l'EFREI (Villejuif), présentant mon parcours, mes compétences et mes réalisations en tant qu'alternant Chargé de Mission IT à la **SEMMARIS** (Marché International de Rungis).

🌐 **[louismorice.fr](https://louismorice.fr)**

---

## Présentation

Ce portfolio regroupe :

- Mon **CV** et mon parcours (BTS SIO SISR 2024-2026, alternance SEMMARIS)
- Les **6 blocs de compétences BTS SIO** documentés avec des réalisations concrètes
- Une **veille technologique** sur la cybersécurité, la conteneurisation et l'administration système
- La présentation de l'**épreuve E5**
- Mes **projets** (administration système, virtualisation, Python, cybersécurité)

---

## Stack technique

- **HTML5 / CSS3** - structure et mise en page
- **Bootstrap 5.3** - composants et grille responsive
- **Font Awesome 6** - icônes
- **Google Fonts** - Inter & Space Grotesk
- **JavaScript vanilla** - animations (IntersectionObserver)

Aucun framework front-end, aucune dépendance NPM. Site statique pur.

## SEO

Chaque page embarque :

- `<meta name="description">` unique et ciblée
- Balises **Open Graph** complètes (titre, description, image, URL, locale)
- **Twitter Cards** (`summary_large_image`)
- Balise `<link rel="canonical">`
- **JSON-LD schema.org** (`Person` + `WebSite` sur l'accueil, `Person` + `WorkExperience` sur le CV)
- Liens sortants sémantiques vers [rungisinternational.com](https://www.rungisinternational.com/decouvrir/missions) et [Wikipedia SEMMARIS](https://fr.wikipedia.org/wiki/Soci%C3%A9t%C3%A9_d%27%C3%A9conomie_mixte_d%27am%C3%A9nagement_et_de_gestion_du_march%C3%A9_d%27int%C3%A9r%C3%AAt_national_de_Rungis)

## Déploiement

Site statique - aucune compilation requise. Il suffit d'héberger les fichiers sur n'importe quel serveur web ou service de pages statiques (GitHub Pages, Netlify, Apache, Nginx…).

```bash
# Cloner le dépôt
git clone https://github.com/BENZOOgataga/ProfesionalPortfolio.git

# Ouvrir directement dans le navigateur
open index.html
```
